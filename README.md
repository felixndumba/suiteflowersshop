# Sweet Petals Market

Premium flowers and gifts e-commerce site built with React, Vite, Tailwind, shadcn/ui.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:8080

## Features

- Product catalog
- Shopping cart
- Gift collections
- Occasions & delivery

## Testing

```bash
npm test
npx playwright test
```
